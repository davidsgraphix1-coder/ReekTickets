# Full Control Tab - Complete Feature Overview

## What is the Full Control Tab?

The **Full Control** tab is the main dashboard in the Admin Panel where the primary admin has complete visibility and control over all ReekTickets operations.

---

## 📊 Full Control Dashboard Sections

### 1. **Statistics Cards** (Top Section)
Quick overview of key metrics:
- **Total Users** - Count of all registered users
- **Total Events** - All events on the platform
- **Total Tickets Sold** - Sum of all tickets ever sold
- **Total Revenue** - Total money earned from all transactions
- **Active Vendors** - Count of vendor accounts
- **Sales Agents** - Count of sales agents on platform

### 2. **Quick Actions** (Button Bar)
Shortcuts to common admin tasks:
- 📢 **Send Announcement** - Broadcast message to all users
- 📊 **Export Data** - Download platform data (users, events, payments)
- 🔍 **System Health** - Check server status and performance
- ⚡ **Platform Status** - View real-time system uptime

### 3. **Platform Revenue Summary** (Highlighted Section)
Sales and payment tracking:
- **Calculated Revenue** - Total computed from all payments
- **Stats Total Revenue** - Total from stats aggregation
- **Total Transactions** - Count of all payment transactions

### 4. **Sales Agents Section**
Manage sales agents:
- View all sales agents on the platform
- See their sales count and commission rates
- Track when they joined
- Status (active/inactive)
- Quick actions: Add new agents

| Column | Purpose |
|--------|---------|
| Agent Name | Display sales agent's full name |
| Email | Agent's email address |
| Phone | Agent's contact number |
| Sales Count | How many transactions they facilitated |
| Commission | Their commission percentage (e.g., 5%) |
| Status | Active, inactive, or suspended |
| Joined | Date they registered as agent |

### 5. **Vendors Section**
Manage all vendor accounts:
- View all vendors and their business details
- Track total revenue generated by each vendor
- See business types and contact info
- Monitor vendor status

| Column | Purpose |
|--------|---------|
| Vendor Name | Business owner's name |
| Business Type | Category (catering, decor, etc.) |
| Email | Vendor's email |
| Phone | Vendor's contact |
| Total Revenue | Sales they've generated |
| Status | Active or inactive |
| Joined | Registration date |

### 6. **System Notifications**
Real-time alerts and updates:
- 🔔 Server alerts
- 🔔 Platform status changes
- 🔔 System maintenance notices
- 🔔 Performance warnings

### 7. **Fraud Detection Alerts**
Security monitoring:
- Suspicious transaction detection
- High-value transaction alerts
- Unusual account activity
- Actions for review and investigation

| Column | Purpose |
|--------|---------|
| Alert Type | Type of fraud detected |
| User | Account involved |
| Details | Specific details of alert |
| Timestamp | When alert was triggered |
| Status | Pending, reviewed, resolved |
| Action | Admin button to review |

### 8. **Admin Activity Logs**
Track what admins do:
- Log all admin actions
- See changes made to the system
- Audit trail for compliance
- Track who did what and when

| Column | Purpose |
|--------|---------|
| Admin | Which admin performed action |
| Action | What they did (created, edited, deleted) |
| Target | What was affected (user, event, ticket) |
| Changes | Specific modifications made |
| Timestamp | When action occurred |

### 9. **User Activity Logs**
Monitor user behavior:
- Track user actions and logins
- See purchase history
- Monitor suspicious behavior
- IP address logging for security

| Column | Purpose |
|--------|---------|
| User | Which user |
| Activity | What they did |
| Type | Payment, ticket purchase, login, etc. |
| IP Address | Their device's IP |
| Timestamp | When it happened |

**Full admin control over customer support chats:**

#### ChatSupport Status & Settings
- **Enable/Disable Toggle** - Turn ChatSupport on/off instantly
- **Active Chat Support Admins Counter** - See how many admins are assigned
- **Add Chat Support Admin Button** - Quickly assign new admins
- **Auto-Response Message** - Customize what users see when no admin is available
- **Save Settings Button** - Apply all changes

#### Assigned Chat Support Admins
- List of all admins with ChatSupport access
- Shows admin name and email
- Quick remove button to revoke access
- Click to add more admins anytime

#### ChatSupport Features List
- ✅ Real-time user-to-admin messaging
- ✅ Chat history and transcripts
- ✅ Multi-admin support (handle chats simultaneously)
- ✅ Admin assignment and load balancing
- ✅ Auto-response for offline periods
- ✅ Chat rating and feedback system
- ✅ Chat search and filtering capabilities
- ✅ User account linking in chat window
- ✅ Admin activity tracking

## 👥 User Roles & Dashboard Access

### Main Admin
- ✅ Access all Full Control features
- ✅ Modify all settings
- ✅ Assign/remove other admins
- ✅ View all sensitive data
- ✅ Cannot be removed from system
- ✅ Full ChatSupport control

### Chat Support Admin
- ✅ View and respond to user chats
- ✅ See chat history
- ❌ Cannot modify ChatSupport settings
- ❌ Cannot add/remove other admins
- ❌ Can be removed by main admin

### Other User Roles
- ❌ Cannot access Admin Dashboard
- ✅ Can contact admins via ChatSupport
- ✅ Can view their own activity

## 🔧 How to Use Each Feature

### Viewing Statistics
1. Go to **Full Control** tab
2. See all stats at the top
3. Numbers update in real-time
4. Use for tracking KPIs

### Sending Announcements
1. Click **"Send Announcement"** button
2. Write announcement message
3. Select recipient groups (all users, vendors, agents, etc.)
4. Schedule or send immediately
5. Track delivery and read rates

### Exporting Data
1. Click **"Export Data"** button
2. Select what to export:
   - Users list (with all details)
   - Events list
   - Payments/transactions
   - Vendor information
   - Tickets data
3. Choose format (CSV or Excel)
4. Download to your computer

### Checking System Health
1. Click **"System Health"** button
2. See metrics:
   - Server uptime
   - Response time
   - Database performance
   - API request rates
   - Memory usage
3. Identify bottlenecks

### Managing Vendors
1. Scroll to **Vendors** section
2. See all vendors and their stats
3. Click on vendor row for details
4. Can edit vendor profile
5. Can suspend or activate vendor

### Managing Sales Agents
1. Scroll to **Sales Agents** section
2. View commission rates and sales count
3. Click **"Add Agent"** to recruit new agents
4. Edit agent details or commission
5. Deactivate underperforming agents

### Handling Fraud Alerts
1. Scroll to **Fraud Detection Alerts**
2. Review alert details
3. Check user's account for suspicious activity
4. Click **"Review"** button
5. Take action (freeze account, request verification, etc.)

### Reviewing Admin Actions
1. Scroll to **Admin Activity Logs**
2. See what each admin has done
3. Look for suspicious changes
4. Verify all modifications were authorized
5. Create audit trail for compliance

### Managing ChatSupport ⭐
1. Scroll to **ChatSupport Management** section (bottom)
2. **Enable/Disable** - Toggle ChatSupport on/off
3. **Add Admin** - Click "+ Add Chat Support Admin"
4. **Select Admin** - Choose from dropdown list
5. **Save Auto-Response** - Type message and click Save
6. **Remove Admin** - Click Remove on admin card to revoke access

## 🚀 Common Workflows

### Onboarding a New Support Admin
1. Go to **Full Control** tab
2. Scroll to ChatSupport section
3. Click **"+ Add Chat Support Admin"**
4. Select the admin's name from dropdown
5. Click **"Add Chat Admin"**
6. Admin now has full ChatSupport access

### Handling High-Value Transaction Alert
1. Fraud alert appears in dashboard
2. Click **"Review"** on the alert
3. Check user's account history
4. Verify purchase is legitimate
5. Mark alert as "resolved" if okay
6. Or freeze account if fraudulent

### Exporting Monthly Report
1. Click **"Export Data"** button
2. Select all data types for the month
3. Download Excel file
4. Open in spreadsheet software
5. Create charts and analysis
6. Share with stakeholders

### Responding to Critical System Alert
1. See notification in dashboard
2. Click system notification
3. View alert details
4. Follow recommended action
5. Contact technical support if needed
6. Update admin logs with actions taken
